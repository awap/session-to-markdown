# Session to Markdown — 개인정보 처리방침 / Privacy Policy

최종 수정: 2026-07-03

## 한국어

Session to Markdown(이하 "확장")은 어떠한 개인정보도 수집, 저장, 전송하지 않습니다.

- 확장은 사용자가 저장 버튼을 누른 시점에만 현재 탭의 대화 내용을 읽습니다.
- 읽은 내용은 브라우저 안에서 Markdown으로 변환되어 사용자의 로컬 다운로드 폴더에 저장됩니다.
- 어떤 데이터도 외부 서버로 전송되지 않으며, 분석 도구·광고·트래커를 포함하지 않습니다.
- 확장은 계정 정보, 쿠키, 브라우징 기록에 접근하지 않습니다.

문의: https://github.com/awap/session-to-markdown/issues

## English

Session to Markdown (the "Extension") does not collect, store, or transmit any personal data.

- The Extension reads the conversation in the current tab only when the user clicks the Save button.
- The content is converted to Markdown entirely within the browser and saved to the user's local Downloads folder.
- No data is ever sent to external servers. The Extension contains no analytics, ads, or trackers.
- The Extension does not access account information, cookies, or browsing history.

Contact: https://github.com/awap/session-to-markdown/issues
